# CartCraze E-Commerce Platform

## Features
- User authentication (login/logout)
- Product browsing and search
- Shopping cart functionality
- Order history tracking
- Responsive design

### Frontend Setup
```bash
git clone https://github.com/yourusername/cartcraze.git
cd cartcraze/client
npm install
npm run dev
```
